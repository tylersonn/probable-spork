# **App Name**: Genesis Landing

## Core Features:

- Hero Section: Displays a clear value proposition with headline, supporting text, and prominent call-to-action buttons.
- Navigation Bar: A consistent navbar for easy site navigation with links to key sections and a distinct logo.
- CTA Buttons: Engaging call-to-action buttons that encourage user interaction and conversion.
- Responsive Layout: Adapts the layout to provide an optimal viewing experience across various screen sizes and devices.
- Mobile Navigation: Implements a hamburger menu for efficient navigation on mobile devices.
- Grid Texture Overlay: Applies a subtle background grid to add depth and visual interest to the landing page.

## Style Guidelines:

- Primary color: Warm orange (#FF6B35), taken from user prompt, is used to create engaging calls-to-action. It evokes excitement and energy.
- Background color: Dark charcoal (#1A1A1A), taken from user prompt, is for the primary background. It provides a sophisticated and modern base for content presentation.
- Accent color: A slightly brighter shade of orange (#FF8C42) is used for hover states and active elements, providing visual feedback and interactivity.
- Body and headline font: 'Inter', a sans-serif font, will maintain a professional, readable style and works for both headlines and body.
- Note: currently only Google Fonts are supported.
- Simple, clean icons for key sections using the Lucide React library.
- Mobile-first approach, content stacks vertically on mobile and horizontally on larger screens.
- Subtle opacity transitions on buttons and interactive elements for smoother user experience.
